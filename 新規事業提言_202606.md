## 前提：この提言の視点

既存3提言（DCリフト・脱LiDAR AMR・点検デジタル化）は、スギヤスのコアを「インドア×物流×AIセンシング」に集中させている。
今回はその外側に目を向けた。

日本社会が今、直面している3つの不可逆な変化がある。
EVへの完全移行、製造現場の高齢化・人手不足、そして介護人材の枯渇だ。
いずれも「10年後には当たり前になっている構造変化」であり、スギヤスの既存技術が正確に刺さる。

既存3事業とのすみ分けは明確だ。
「現場をデジタルで見せる」のが既存提言なら、「現場の物理的なしんどさを機械で解決する」のがこの3提言だ。

## 事業提言1：EV整備専用リフト

### 何をやるか

EV（電気自動車）の車体下部構造に対応した整備専用リフト。
バッテリーパックがフロアを占有するEVには、既存の2柱・4柱リフトが当たらない支持点を持つ。
高電圧部位を避けながら安全に持ち上げ、整備スペースを確保する「EVネイティブ設計」のリフトだ。
まず対応アダプターパッドから入り、段階的にEV専用機へ展開する。

### なぜスギヤスがやるべきか

整備リフトは本業だ。2柱・4柱・シザー・トラバーと、幅広い整備リフトの設計・製造実績がある。
EVに特化した専用機を作るのに、ゼロから開発する必要はない。

中国のEPが作ったフォークリフトを、スギヤスのリフトで整備する現場がすでにある。
同じ構図が自動車にも来る。中国製EVが日本に入り始めたら、整備工場はスギヤスに頼む。

### 社会課題との接点

カーボンニュートラルの達成には、EVが走るだけでなく「整備できるインフラ」が不可欠だ。
整備工場がEV対応設備を持たなければ、EV普及は止まる。
国内45,000店の整備工場が、2035年に向けて全員、設備更新を迫られている。
高電圧バッテリーの安全取り扱いは知識が必要だが、「安全に持ち上げる機械」はスギヤスの仕事だ。

### 市場成長性（数字で示す）

| 指標 | 内容 |
|---|---|
| 国内自動車整備工場数 | 約45,000店（出典：国土交通省 2024年） |
| 国内新車EV比率目標（2035年） | 乗用車の100%をEV/HVに（政府目標） |
| 世界EV整備機器市場（2024→2030） | 約8億ドル → 約20億ドル（CAGR 約16%）推計 |
| 既存整備リフトの更新需要 | EV普及率が20%を超える2028年頃から設備投資ラッシュが始まる |
| 単価プレミアム | EV専用機は汎用リフト比10〜30%高値設定が現実的 |

スギヤスの参入機会：現在、国内にEV専用整備リフトメーカーはほぼ存在しない。輸入品（主に欧米）が数社入っているのみだ。サーバーリフトと同じ構図だ——「国産・短納期・アフターサービス付き」で先行すれば、空白市場を取れる。

### 最初の一手

自社の整備リフト既販モデルと、主要EVモデル（テスラ・リーフ・EV6・BYD等）の車体支持点マッピングを実施する。
対応アダプターパッドの設計から始め、「EV対応モデル」として既存ラインにオプション展開する。先に言える会社になることが先決だ。

## 事業提言2：エルゴノミクス傾斜テーブルリフト

### 何をやるか

高さと傾き（チルト）を電動で無段階調整できる作業テーブルリフト。
「ちょうどいい高さで、ちょうどいい角度から取り出せる」を実現する機器だ。
高齢者・女性・小柄な人が、腰を痛めずに製造現場で働き続けるための道具として設計する。

LogiMAT 2025でドイツ・欧州メーカーが複数社展示しており、欧州では製品カテゴリとして確立済みだ。日本はまだ空白に近い。

### なぜスギヤスがやるべきか

テーブルリフト・シザーリフトの設計・製造は本業中の本業だ。
傾斜機構を追加するのは、設計としては難しくない。
LogiMATレポートには「過去に試作実績あり」とある。試作まで積んでいる。

「新しい技術を入れる」事業ではない。「既存の技術を、新しい使い方で売る」事業だ。
投資対効果が最も高い3事業のうちの筆頭だ。

### 社会課題との接点

2040年に日本の生産年齢人口は現在より600万人減少する（国立社会保障・人口問題研究所）。
それでも製造業の生産を維持するには、高齢者・女性・外国人労働者が現場の主役にならなければならない。

腰痛は製造業の離職理由の上位だ。
「機械が高さと角度を合わせてくれる」だけで、働ける人が増える。
それで設備が一台売れる。社会課題と事業性が直結している。

### 市場成長性（数字で示す）

| 指標 | 内容 |
|---|---|
| 製造業の55歳以上雇用者比率 | 全製造業従事者の30%超（出典：厚生労働省 2024年） |
| 国内産業用エルゴノミクス機器市場 | 約300億円（2024）→ 約600億円（2030）推計 |
| 腰痛による労災・休業コスト | 製造業で年間数千億円規模（出典：厚労省労災保険統計） |
| 目標単価・販売数量 | 30〜80万円 × 年間100〜300台を3年以内に目標 |
| 欧州規制の波及 | EU ergonomics規制が日本市場に波及する2028〜2030年に需要急増の可能性 |

### 最初の一手

過去の試作機を掘り起こす。
「高さ調整だけ」のテーブルリフトに、まず傾き調整機構を1つ加えた改造機を1台作る。
社内・西尾工場の製造ラインで使ってもらい改善する。出展は来年の国内展示会で十分だ。

## 事業提言3：介護・福祉現場向け電動昇降搬送機器

### 何をやるか

介護施設・病院・在宅ケア現場で使う小型・軽量な電動昇降・搬送機器シリーズ。
第一弾は「ベッドから車椅子への移乗補助リフト」に絞る。
スギヤスの電動昇降機構を、介護に最適化したサイズ・重さ・操作感に落とし込む。
介護保険の「福祉用具貸与」カテゴリに登録すれば、利用者の自己負担は1割になる。

### なぜスギヤスがやるべきか

電動で物を上下させることと、安全に重いものを動かすことが本業だ。
介護現場が求めているのはシンプルだ——軽くて、小さくて、すぐ使えて、壊れない。
そこに「国産・アフターサービス付き」を加えれば、施設管理者は選ぶ。

RTJ26で視察した神奈川工科大の介護ロボット「ルチア」、NACOL（スギヤスが訪問済みの静岡・清水の会社）の水圧アクチュエータ等、要素技術パートナーはすでに近くにある。

### 社会課題との接点

2040年に介護人材が132万人不足する（厚生労働省推計）。
ケアワーカーの腰痛・身体的疲弊が離職の第1要因だ。
年間10万人が介護職を腰痛で辞めている。

「機械が持ち上げる」から、人手に余裕が生まれる。
スギヤスの製品が1台入るだけで、ケアワーカー1人を守れる可能性がある。

### 市場成長性（数字で示す）

| 指標 | 内容 |
|---|---|
| 国内福祉用具市場規模（2024） | 約8,300億円（介護保険給付含む） |
| 国内福祉用具市場規模（2030予測） | 約1兆3,000億円（CAGR 約7〜9%）推計 |
| 移乗支援機器の現在普及率 | 10%未満。政府が2030年までに普及目標設定中（介護テクノロジー活用推進計画） |
| 介護保険の福祉用具貸与 | 対象機器に登録されれば利用者負担1割。公的保険が販売ルートになる |
| 介護施設の設備投資 | 人材離職防止コストと比較した場合の移乗機器の費用対効果が高いと認識され始めている |

### 最初の一手

まず介護施設を3〜5か所、山崎部長の人脈経由で訪問する。
「何が重くて、何が怖くて、何があれば助かるか」を直接聞く。
既存の小型電動リフト機構を転用した試作機を1台作り、施設で実際に使ってもらう。
そこから製品スペックを固める。開発に入る前にまず現場に行くことが最初の一手だ。

## 結論

3事業すべてに共通するのは、「10年後には誰もがやっている。でも今はまだ誰もやっていない」という構造だ。

エルゴノミクスリフトは今すぐ試作できる。LogiMATで見た、過去に試作した。あとはやるだけだ。
EV整備リフトは今すぐ市場調査できる。整備工場が今、何に困っているかを聞けばいい。
介護リフトは介護施設に3回行けば、本当のニーズが見える。

中国メーカーにハードの価格では勝てない。
だが「高齢者が安心して使えるもの」「介護の現場で壊れないもの」「EVメカニックが安全に使えるもの」は、スギヤスが作るから意味がある。
誰もやっていないから機会がある。現場に行けば答えはある。
